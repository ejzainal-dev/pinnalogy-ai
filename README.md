# Pinnalogy AI - Ear Anatomy Analysis

A web application for automatic ear anatomy segmentation using deep learning.

## Features
- Automatic ear anatomy segmentation
- Four region detection: Helix, Antihelix, Concha, Lobule
- Quantitative analysis
- Export results
